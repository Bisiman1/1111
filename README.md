# 图书阅读管理微信小程序服务器端
## 微信小程序服务器端
是由SpringBoot2.1 + MyBatis +Mysql5.6等技术实现的项目

### 技术栈
#### 后端
* 核心框架：SpringBoot
* 持久层框架：MyBatis

* 下载源码，导入本地数据库sql文件，然后修改application-dev.yml文件中连接数据库的用户名、密码。运行项目即可。
